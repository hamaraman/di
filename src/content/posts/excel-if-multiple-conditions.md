IF 함수는 조건 하나만 판단합니다. 조건이 여러 개일 때는 방법이 몇 가지 있는데, 상황에 맞게 고르면 됩니다.

## 기본 IF 함수 복습

```
=IF(조건, 참일_때_값, 거짓일_때_값)
```

예: `=IF(A2>=60, "합격", "불합격")`

## 방법 1. 중첩 IF (조건을 단계별로)

점수를 등급으로 나누듯, 조건을 순서대로 판단할 때 씁니다. IF 안에 IF를 넣습니다.

```
=IF(A2>=90, "A", IF(A2>=80, "B", IF(A2>=70, "C", "F")))
```

앞 조건부터 차례로 검사해 처음 맞는 값을 반환합니다. 다만 조건이 많아지면 괄호가 복잡해집니다.

## 방법 2. AND·OR로 여러 조건 동시에

**여러 조건을 모두** 만족해야 하면 `AND`, **하나라도** 만족하면 `OR`를 IF와 조합합니다.

```
=IF(AND(A2>=60, B2>=60), "통과", "재시험")
=IF(OR(A2>=90, B2>=90), "우수", "일반")
```

- `AND(조건1, 조건2, …)`: 전부 참일 때만 참
- `OR(조건1, 조건2, …)`: 하나라도 참이면 참

## 방법 3. IFS 함수로 간단하게 (엑셀 2019·365)

중첩 IF가 길어질 때 `IFS`를 쓰면 훨씬 읽기 쉽습니다.

```
=IFS(A2>=90, "A", A2>=80, "B", A2>=70, "C", TRUE, "F")
```

조건과 값을 쌍으로 나열하며, 마지막 `TRUE, "F"`는 "그 외 전부"를 뜻합니다. 구버전 엑셀에는 IFS가 없으니 그때는 방법 1을 씁니다.

## 자주 묻는 질문

**Q. 중첩 IF는 몇 개까지 되나요?**
엑셀은 IF를 최대 64개까지 중첩할 수 있지만, 7~8개만 넘어도 관리가 어렵습니다. 그럴 땐 IFS나 VLOOKUP 표 참조로 바꾸는 게 낫습니다.

**Q. 값을 찾아오는 조건 분기는 IF보다 VLOOKUP이 낫나요?**
"이 값이면 저 값" 식의 매칭이 많으면 IF보다 표 참조가 깔끔합니다. VLOOKUP 사용법과 오류 해결은 [엑셀 VLOOKUP #N/A 오류 해결 방법](/posts/excel-vlookup-na-error/)에서 다뤘습니다.
