VLOOKUP은 표에서 원하는 값을 찾아오는 함수입니다. 이름이나 사번을 넣으면 그 옆의 가격, 부서 같은 값을 자동으로 가져옵니다. 처음이라면 아래 순서대로 따라오면 됩니다.

## VLOOKUP이 하는 일

한 표에서 **기준이 되는 값**을 찾아, 그 값과 **같은 행에 있는 다른 열의 값**을 돌려주는 함수입니다. 예를 들어 상품 목록에서 상품코드로 가격을 찾아올 때 씁니다. 세로(열) 방향으로 값을 찾는다고 해서 V(Vertical)LOOKUP입니다.

기본 문법은 다음과 같습니다.

```
=VLOOKUP(찾을값, 표범위, 열번호, FALSE)
```

## 1. 인수 4개 이해하기

VLOOKUP은 괄호 안에 값 4개를 순서대로 넣습니다.

- **찾을값**: 표에서 찾고 싶은 기준값입니다. 예: 상품코드가 들어 있는 `A2` 셀.
- **표범위**: 값을 찾을 표 전체입니다. 이때 **찾을값은 반드시 범위의 맨 왼쪽 첫 번째 열**에 있어야 합니다.
- **열번호**: 표범위에서 몇 번째 열의 값을 가져올지 숫자로 적습니다. 맨 왼쪽 열이 1번입니다.
- **일치옵션**: `FALSE`(정확히 일치) 또는 `TRUE`(근사 일치)입니다. 초보자는 거의 항상 `FALSE`를 씁니다.

## 2. 단계별 예제

`D`열에 상품코드, `E`열에 가격이 있는 표(`D2:E100`)에서, `A2`에 입력한 상품코드의 가격을 찾아온다고 해봅시다.

1. 결과를 표시할 빈 셀을 클릭합니다.
2. `=VLOOKUP(` 를 입력합니다.
3. 찾을값으로 `A2` 를 클릭하고 쉼표를 찍습니다.
4. 표범위로 `D2:E100` 을 드래그하고 쉼표를 찍습니다.
5. 가격은 표에서 2번째 열이므로 열번호에 `2` 를 넣고 쉼표를 찍습니다.
6. 마지막에 `FALSE` 를 넣고 괄호를 닫습니다.

완성된 수식은 이렇습니다.

```
=VLOOKUP(A2, D2:E100, 2, FALSE)
```

수식을 아래로 복사해 여러 행에 적용할 계획이라면, 표범위가 밀리지 않도록 `$`로 고정하는 것이 좋습니다.

```
=VLOOKUP(A2, $D$2:$E$100, 2, FALSE)
```

## 3. 자주 하는 실수

- **찾을값이 첫 번째 열에 없음**: VLOOKUP은 표범위의 맨 왼쪽 열에서만 값을 찾습니다. 찾을 기준값이 왼쪽에 오도록 범위를 잡으세요.
- **일치옵션을 빠뜨림**: 네 번째 인수를 생략하면 근사 일치로 동작해 엉뚱한 값이 나올 수 있습니다. 정확한 값을 찾을 땐 꼭 `FALSE`를 넣습니다.
- **열번호를 시트 기준으로 셈**: 열번호는 시트의 A·B·C가 아니라 **표범위 안에서** 몇 번째 열인지로 셉니다.
- **숫자와 텍스트 형식 불일치**: 찾을값은 숫자인데 표의 값이 텍스트로 저장돼 있으면 안 맞습니다. 값이 텍스트로 저장돼 계산이 안 맞을 때는 [엑셀 수식 계산이 안 될 때](/posts/excel-formula-not-calculating/)를 참고하세요.

값이 분명히 있는데도 `#N/A`가 뜬다면 원인이 몇 가지로 정해져 있습니다. [엑셀 VLOOKUP #N/A 오류 원인과 해결](/posts/excel-vlookup-na-error/)에서 순서대로 확인해 보세요.

## 자주 묻는 질문

**Q. 열번호는 어떻게 세나요?**
시트의 열 이름(A, B, C)이 아니라 **표범위 안에서** 왼쪽부터 1, 2, 3으로 셉니다. 표범위가 `D2:E100`이면 D열이 1번, E열이 2번입니다.

**Q. TRUE와 FALSE 중 뭘 써야 하나요?**
정확히 일치하는 값을 찾으려면 `FALSE`(또는 `0`)를 씁니다. `TRUE`는 구간별 등급표처럼 근사값을 찾을 때만 쓰며, 이때는 첫 열이 오름차순으로 정렬돼 있어야 합니다. 초보자는 `FALSE`로 시작하세요.

**Q. 조건을 여러 개 따져서 값을 찾고 싶어요.**
"점수가 60 이상이면서 출석도 채웠으면" 같은 여러 조건 판단은 VLOOKUP이 아니라 IF 계열이 맞습니다. [엑셀 IF 함수 다중 조건 사용법](/posts/excel-if-multiple-conditions/)에서 AND·OR·IFS 조합을 다뤘습니다.

**Q. VLOOKUP은 왼쪽 값도 찾아오나요?**
기본 VLOOKUP은 찾을값의 **오른쪽 열**만 가져올 수 있습니다. 왼쪽 열 값이 필요하면 INDEX·MATCH 조합이나 최신 버전의 XLOOKUP을 쓰는 것이 편합니다.
