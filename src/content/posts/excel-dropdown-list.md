셀을 클릭하면 정해진 항목 중에서 고르는 **드롭다운 목록**은 데이터 유효성 검사로 만듭니다. 오타를 막고 입력을 통일할 수 있어 실무에서 자주 씁니다.

## 방법 1. 항목을 직접 입력해서 만들기

목록이 짧고 잘 안 바뀔 때 쉽습니다.

1. 드롭다운을 넣을 셀(또는 범위)을 선택합니다.
2. **데이터** 탭 → **데이터 유효성 검사**를 클릭합니다.
3. **제한 대상**을 **목록**으로 선택합니다.
4. **원본** 칸에 항목을 쉼표로 구분해 입력합니다. 예: `대기,진행중,완료`
5. 확인을 누르면 셀에 드롭다운 화살표가 생깁니다.

## 방법 2. 셀 범위를 참조해서 만들기

목록이 길거나 자주 바뀔 때 좋습니다. 목록을 시트 어딘가에 적어두고 그 범위를 참조합니다.

1. 빈 곳(예: H열)에 항목들을 세로로 입력해 둡니다.
2. 위와 같이 **데이터 유효성 검사 → 목록**을 선택합니다.
3. **원본** 칸을 클릭한 뒤 항목이 적힌 범위를 드래그합니다. 예: `=$H$1:$H$5`

이 방식은 원본 범위의 값만 고치면 드롭다운도 같이 바뀝니다.

## 드롭다운 수정·삭제하기

- **항목 변경**: 같은 셀에서 다시 **데이터 유효성 검사**를 열어 원본을 수정합니다.
- **드롭다운 삭제**: 셀 선택 후 데이터 유효성 검사 창에서 **모두 지우기**를 클릭합니다.

## 자주 묻는 질문

**Q. 드롭다운 화살표가 안 보여요.**
셀을 클릭했을 때만 오른쪽에 화살표가 나타납니다. 그래도 없다면 유효성 검사 창에서 **셀 안에서 이동 표시(드롭다운 표시)** 옵션이 켜져 있는지 확인하세요.

**Q. 목록에 없는 값도 입력되게 하고 싶어요.**
데이터 유효성 검사의 **오류 메시지** 탭에서 스타일을 '경고'나 '정보'로 바꾸면, 목록 외 값도 확인 후 입력할 수 있습니다. 기본값인 '중지'는 목록 외 입력을 막습니다.
