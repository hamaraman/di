## 1. COUNTIF — 조건에 맞는 셀 개수 세기

특정 조건을 만족하는 셀이 **몇 개인지** 셀 때 씁니다. 문법은 인수 두 개로 단순합니다.

```
=COUNTIF(범위, 조건)
```

- **범위**: 셀 것을 찾을 셀 묶음입니다.
- **조건**: 어떤 값을 셀지 정하는 기준입니다.

예를 들어 `B2:B100`에 점수가 있고 60점 이상인 사람 수를 세려면 이렇게 씁니다.

```
=COUNTIF(B2:B100, ">=60")
```

`">=60"`처럼 부등호가 들어간 조건은 반드시 큰따옴표로 감쌉니다. `"사과"`처럼 특정 값과 정확히 같은 개수를 셀 때도 큰따옴표를 씁니다.

```
=COUNTIF(A2:A100, "사과")
```

## 2. SUMIF — 조건에 맞는 값 합계 구하기

개수가 아니라 **합계**가 필요하면 SUMIF입니다. 여기서 인수 순서가 COUNTIF와 달라 헷갈리기 쉬우니 주의하세요.

```
=SUMIF(조건범위, 조건, 합계범위)
```

- **조건범위**: 조건을 검사할 셀 묶음입니다.
- **조건**: 어떤 행을 더할지 정하는 기준입니다.
- **합계범위**: 실제로 더할 값이 있는 셀 묶음입니다.

`A`열에 상품명, `C`열에 금액이 있을 때, "사과"의 금액만 모두 더하려면 이렇게 씁니다.

```
=SUMIF(A2:A100, "사과", C2:C100)
```

즉 "A열에서 사과인 행을 찾아, 그 행의 C열 값을 더한다"는 뜻입니다. 조건을 검사하는 열과 더하는 열이 같다면 합계범위는 생략할 수 있습니다. 아래는 `B`열에서 100 이상인 값만 골라 그대로 더합니다.

```
=SUMIF(B2:B100, ">=100")
```

## 3. 와일드카드로 "특정 문자 포함" 세기

정확히 일치하는 값이 아니라 **어떤 글자가 들어간** 셀을 세고 싶을 때는 와일드카드를 씁니다.

- `*` : 글자 수 상관없이 아무 문자열 (예: 서울을 포함)
- `?` : 딱 한 글자

주소 `A`열에서 "서울"이 들어간 셀이 몇 개인지 세려면 이렇게 씁니다.

```
=COUNTIF(A2:A100, "*서울*")
```

앞에만 오면 `"서울*"`, 뒤에만 오면 `"*서울"`로 조절합니다. 이 와일드카드는 SUMIF의 조건에서도 똑같이 쓸 수 있습니다.

값이 몇 번 나오는지 세어 중복을 찾는 것도 COUNTIF의 대표 쓰임새입니다. 이 방법은 [엑셀 중복값 찾기·제거](/posts/excel-remove-duplicates/)에서 더 자세히 다뤘습니다.

## 4. 조건이 여러 개면 COUNTIFS·SUMIFS

조건을 **두 개 이상** 겹쳐야 할 때는 뒤에 `S`가 붙은 COUNTIFS, SUMIFS를 씁니다.

```
=COUNTIFS(범위1, 조건1, 범위2, 조건2, …)
=SUMIFS(합계범위, 범위1, 조건1, 범위2, 조건2, …)
```

"지역이 서울이면서 금액이 100 이상인" 건수와 금액 합계는 이렇게 구합니다.

```
=COUNTIFS(A2:A100, "서울", C2:C100, ">=100")
=SUMIFS(D2:D100, A2:A100, "서울", C2:C100, ">=100")
```

여기서 꼭 짚어둘 점이 있습니다. **SUMIFS는 합계범위가 맨 앞**에 오지만, SUMIF는 합계범위가 맨 뒤에 옵니다. 함수 이름은 비슷해도 인수 순서가 반대라서 가장 많이 틀리는 부분입니다.

## 5. 자주 하는 실수

- **SUMIF와 SUMIFS의 순서 혼동**: 위에서 말한 대로 SUMIF는 `(조건범위, 조건, 합계범위)`, SUMIFS는 `(합계범위, 조건범위, 조건…)`입니다. 반대로 넣으면 엉뚱한 값이 나옵니다.
- **부등호 조건에 큰따옴표를 안 씀**: `>=60`이 아니라 `">=60"`처럼 따옴표로 감싸야 합니다.
- **COUNTIF와 SUMIF 인수 순서 헷갈림**: COUNTIF는 `(범위, 조건)`, SUMIF는 `(조건범위, 조건, 합계범위)`로 조건의 위치가 다릅니다.
- **범위 크기가 서로 다름**: SUMIF·SUMIFS에서 조건범위와 합계범위의 행 수가 다르면 결과가 어긋날 수 있으니 같은 크기로 맞춥니다.

## 자주 묻는 질문

**Q. COUNTIF와 SUMIF는 언제 뭘 쓰나요?**
"몇 개"인지 셀 때는 COUNTIF, "합이 얼마"인지 더할 때는 SUMIF입니다. 개수는 COUNT, 합계는 SUM이라고 기억하면 헷갈리지 않습니다.

**Q. 조건에 다른 셀 값을 넣어도 되나요?**
됩니다. `=COUNTIF(A2:A100, E1)`처럼 조건 자리에 셀을 참조하면 됩니다. 부등호와 셀을 합칠 때는 `">="&E1`처럼 큰따옴표와 `&`로 이어 붙입니다.

**Q. 조건을 여러 개로 판단해 다른 값을 표시하고 싶어요.**
합계·개수가 아니라 "조건에 따라 다른 결과를 표시"하려는 거라면 IF 계열이 맞습니다. [엑셀 IF 함수 다중 조건 사용법](/posts/excel-if-multiple-conditions/)에서 AND·OR·IFS 조합을 다뤘습니다.

**Q. 조건에 맞는 다른 열의 값을 가져오고 싶어요.**
더하거나 세는 게 아니라 "그 값에 해당하는 다른 항목을 찾아오는" 조회라면 VLOOKUP이 맞습니다. [엑셀 VLOOKUP 사용법 기초](/posts/excel-vlookup-basic/)를 참고하세요.
