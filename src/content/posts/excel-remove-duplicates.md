명단이나 주문 목록을 정리하다 보면 중복된 데이터를 찾아야 할 때가 많습니다. 목적에 따라 세 가지 방법을 골라 쓰면 됩니다.

## 방법 1. 조건부 서식으로 중복값 강조 (지우지 않고 확인만)

원본을 그대로 두고 어디가 겹치는지 눈으로 확인하고 싶을 때 씁니다.

1. 범위를 선택합니다.
2. **홈 → 조건부 서식 → 셀 강조 규칙 → 중복 값**을 클릭합니다.
3. 색을 고르고 확인하면 중복된 셀이 색으로 표시됩니다.

셀 하나가 아니라 중복된 행 전체를 색칠하고 싶다면 수식 규칙이 필요합니다. [엑셀 조건부서식 수식으로 조건에 맞는 행 전체 색칠하기](/posts/excel-conditional-format-highlight-row/)에서 COUNTIF를 활용한 방법을 다뤘습니다.

## 방법 2. '중복된 항목 제거'로 바로 삭제

실제로 중복 행을 지워 하나만 남기고 싶을 때 씁니다.

1. 데이터 범위를 선택합니다.
2. **데이터 → 중복된 항목 제거**를 클릭합니다.
3. 어떤 열을 기준으로 중복을 판단할지 체크한 뒤 확인합니다.

> 주의: 이 기능은 되돌리기 전에는 복구가 어렵습니다. 원본이 걱정되면 시트를 복사해두고 실행하세요.

## 방법 3. COUNTIF로 몇 번 나왔는지 표시

각 값이 몇 번 등장하는지 세거나, 중복 여부를 별도 열에 남기고 싶을 때 씁니다. COUNTIF로 조건별 개수를 세고 SUMIF로 합계까지 구하는 법은 [엑셀 COUNTIF 사용법·SUMIF 조건별 합계](/posts/excel-countif-sumif/)에 정리했습니다.

```
=COUNTIF($A$2:$A$100, A2)
```

결과가 2 이상이면 중복입니다. 첫 번째 항목만 표시하려면 아래처럼 씁니다.

```
=IF(COUNTIF($A$2:A2, A2)>1, "중복", "")
```

여기서 쓴 `IF`처럼 조건을 여러 개 겹쳐 판단하고 싶다면 [엑셀 IF 함수 다중 조건 사용법](/posts/excel-if-multiple-conditions/)을 참고하세요.

## 엑셀 365라면: UNIQUE 함수

Microsoft 365에서는 `=UNIQUE(A2:A100)` 한 줄로 중복을 뺀 고유값 목록을 바로 뽑을 수 있습니다. 이렇게 뽑은 고유값 목록은 [엑셀 드롭다운 목록 만들기](/posts/excel-dropdown-list/)의 원본으로 그대로 쓰기 좋습니다.

## 자주 묻는 질문

**Q. 대소문자를 구분해서 중복을 찾을 수 있나요?**
기본 기능은 대소문자를 구분하지 않습니다. 구분이 필요하면 `EXACT` 함수를 조합해야 합니다.

**Q. 중복을 지웠더니 데이터가 이상하게 섞였어요.**
'중복된 항목 제거'는 선택한 열 기준으로 행 전체를 지웁니다. 한 열만 선택하고 실행하면 다른 열과의 짝이 어긋날 수 있으니, 표 전체를 선택한 뒤 기준 열을 지정하세요.
