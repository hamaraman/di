셀에 "1-2"나 "3/4" 같은 값을 입력했는데 엔터를 누르자마자 "2026-01-02"처럼 날짜로 바뀌는 경우가 있습니다. 엑셀이 하이픈(-)이나 슬래시(/)가 들어간 숫자를 날짜 형식으로 자동 인식하기 때문입니다. 아래 방법으로 막거나 되돌릴 수 있습니다.

## 1. 입력 전 셀 서식을 '텍스트'로 바꾸기 (가장 확실함)

값을 입력하기 **전에** 미리 서식을 지정하는 방법입니다.

1. 대상 셀(또는 범위)을 선택합니다.
2. `Ctrl` + `1`을 눌러 **셀 서식** 창을 엽니다.
3. **표시 형식** 탭에서 **텍스트**를 선택하고 확인을 누릅니다.
4. 이제 값을 입력하면 날짜로 바뀌지 않고 입력한 그대로 저장됩니다.

이미 데이터를 다 입력한 뒤에 서식만 텍스트로 바꾸면 **기존 셀에는 적용되지 않습니다.** 서식을 먼저 바꾸고 다시 입력(또는 재입력)해야 합니다.

## 2. 작은따옴표(')를 앞에 붙이기

한두 개만 일회성으로 입력할 때 편한 방법입니다.

```
'1-2
```

이렇게 값 앞에 작은따옴표를 붙이고 입력하면 엑셀이 이를 텍스트로 인식해 날짜로 바꾸지 않습니다. 셀에는 작은따옴표가 보이지 않고 값만 그대로 남습니다.

## 3. 이미 날짜로 바뀐 값을 되돌리기

이미 입력해서 날짜로 변환된 셀이 많다면, 하나씩 다시 치는 대신 다음 방법을 씁니다.

1. **홈** 탭 → 표시 형식 드롭다운에서 **텍스트**로 바꿉니다. (단, 이미 날짜로 저장된 값은 서식만 바꿔도 원래 숫자로 자동 복원되지 않는 경우가 많습니다.)
2. 원래 입력했던 값 형태로 되돌리려면 빈 열에 `=TEXT(A1,"m-d")` 처럼 원하는 형식의 텍스트 수식을 넣어 값을 뽑아낸 뒤, 그 결과를 복사해서 **값 붙여넣기**로 원래 위치에 덮어씁니다.
3. 데이터가 여러 열에 걸쳐 있다면 **데이터** 탭 → **텍스트 나누기**를 실행하고, 3단계에서 열 데이터 서식을 **텍스트**로 지정하는 방법도 같은 효과를 냅니다.

## 4. 분수를 분수로 유지하려면

"1/2"를 분수로 쓰고 싶은데 날짜로 바뀐다면, 정수부를 포함해 **"0 1/2"**처럼 0과 공백을 붙여 입력하세요. 정수부가 없는 순수 분수는 반드시 앞에 `0`과 띄어쓰기를 넣어야 날짜가 아닌 분수로 인식됩니다.

## 자주 묻는 질문

**Q. 외부에서 데이터를 붙여넣기(복사)할 때도 날짜로 바뀌나요?**
네, 붙여넣을 대상 셀의 서식이 '일반'이면 소스 데이터 형태에 따라 날짜로 자동 인식될 수 있습니다. 붙여넣기 전에 대상 범위를 텍스트 서식으로 미리 지정해두면 방지할 수 있습니다.

**Q. 수식을 텍스트 서식 셀에 입력하면 계산이 안 되던데요?**
맞습니다. 텍스트 서식 셀에서는 수식이 계산되지 않고 글자 그대로 표시됩니다. 관련 증상과 해결법은 [엑셀 수식 안됨·자동 계산 안될 때 해결 방법](/posts/excel-formula-not-calculating/) 글의 2번 항목을 참고하세요.

**Q. 텍스트로 저장한 숫자를 나중에 계산에 쓰고 싶으면요?**
`=VALUE(A1)` 함수로 텍스트를 숫자로 변환해 계산할 수 있습니다.
