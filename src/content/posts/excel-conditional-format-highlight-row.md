재고 목록에서 "품절"인 행 전체를 빨갛게 표시하거나, 마감일이 지난 행을 통째로 강조하고 싶을 때가 있습니다. 그런데 조건부 서식의 기본 메뉴(셀 강조 규칙)는 조건에 맞는 **셀 하나에만** 서식을 입힙니다. 행 전체를 칠하려면 **수식을 사용하는 규칙**이 따로 필요합니다.

## 기본 규칙으로는 왜 행 전체가 안 될까

**홈 → 조건부 서식 → 셀 강조 규칙**에 있는 메뉴들(중복 값, 보다 큼 등)은 검사한 그 셀 자체에만 서식을 적용합니다. C열이 "품절"인지만 검사해서 C열만 칠하는 식입니다. 행 전체(A~E열 등)에 서식을 입히려면 **새 규칙 → 수식을 사용하여 서식을 지정할 셀 결정**으로 직접 조건식을 걸어야 합니다.

## 단계별로 따라하기

1. 서식을 적용할 범위를 선택합니다. 헤더는 제외하고 데이터 영역 전체를 잡습니다. 예: `A2:E100`
2. **홈 → 조건부 서식 → 새 규칙**을 클릭합니다.
3. **수식을 사용하여 서식을 지정할 셀 결정**을 선택합니다.
4. 수식 입력란에 조건을 넣습니다. 상태가 C열에 있고 "품절"이면 행 전체를 칠하고 싶다면:

```
=$C2="품절"
```

5. **서식** 버튼을 눌러 채우기 색을 정하고 확인을 누릅니다.

핵심은 `$C2`처럼 **열만 고정하고 행은 고정하지 않는 것**입니다. 범위를 A2:E100으로 잡고 이 수식을 넣으면, 각 행마다 "그 행의 C열" 값을 검사해 조건이 참이면 행 전체(A~E열)에 서식이 적용됩니다.

## 숫자 조건 예제

금액이 담긴 D열이 10만 원을 초과하는 행 전체를 강조하고 싶다면:

```
=$D2>100000
```

## 여러 조건을 동시에 (AND·OR)

마감일(E열)이 지났는데 상태(F열)가 아직 "완료"가 아닌 행만 강조하고 싶다면 `AND`를 함께 씁니다.

```
=AND($E2<TODAY(), $F2<>"완료")
```

조건을 여러 개 겹쳐 판단하는 원리는 IF 함수와 동일합니다. AND·OR 조합과 IFS 함수까지는 [엑셀 IF 함수 다중 조건 사용법](/posts/excel-if-multiple-conditions/)에서 더 자세히 다뤘습니다.

## COUNTIF와 조합해 중복 행 전체 강조

이름(A열)이 중복된 행 전체를 강조하고 싶다면 COUNTIF를 조건식으로 씁니다.

```
=COUNTIF($A$2:$A$100, $A2)>1
```

셀 하나만 강조하는 기본 "중복 값" 규칙과의 차이, 그리고 중복을 아예 지우는 방법은 [엑셀 중복값 찾기·제거하는 3가지 방법](/posts/excel-remove-duplicates/)에 정리해 두었습니다. COUNTIF로 조건별 개수를 세는 기본 문법은 [엑셀 COUNTIF 사용법·SUMIF 조건별 합계](/posts/excel-countif-sumif/)를 참고하세요.

## 자주 하는 실수

- **`$` 위치 실수**: `$C$2`처럼 행까지 고정하면 모든 행이 2행의 조건 결과만 따라가서, 전부 같은 색이 되거나 전혀 안 칠해집니다. 열만 고정(`$C2`)하고 행은 고정하지 마세요.
- **범위를 조건 열만 선택**: A2:E100이 아니라 C2:C100만 선택하고 규칙을 걸면 C열만 칠해지고 나머지 열은 그대로입니다. 서식을 입힐 열을 전부 포함해서 범위를 잡아야 합니다.
- **규칙이 여러 개 겹쳐 꼬임**: 조건부 서식 규칙이 여러 개 걸려 있으면 위에 있는 규칙이 우선 적용됩니다. 원하는 대로 안 보이면 **홈 → 조건부 서식 → 규칙 관리**에서 순서와 겹치는 범위를 확인하세요.

## 자주 묻는 질문

**Q. 규칙이 여러 개 겹치면 어떻게 되나요?**
**홈 → 조건부 서식 → 규칙 관리**에서 위쪽에 있는 규칙이 우선 적용됩니다. 서로 다른 서식(채우기색 vs 글자색)을 조합하면 규칙이 겹쳐도 충돌을 줄일 수 있습니다.

**Q. 다른 시트의 값을 조건으로 쓸 수 있나요?**
같은 통합 문서 안이라면 가능합니다. 수식에 `시트명!C2`처럼 시트명을 정확히 포함해야 하며, 시트를 옮기거나 이름을 바꾸면 수식이 깨질 수 있으니 주의하세요.

**Q. 정확히 일치가 아니라 특정 단어가 포함되면 색칠하고 싶어요.**
`SEARCH`와 `ISNUMBER`를 조합합니다.

```
=ISNUMBER(SEARCH("지연", $C2))
```

`SEARCH`는 대소문자를 구분하지 않고, 셀 안 어디에 단어가 있어도 찾아냅니다.
